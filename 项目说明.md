# 弈途微信小程序 Demo 项目说明

## 项目完成情况

✅ 已完成基于Vue 3技术栈的弈途微信小程序demo开发，用于向客户展示交互流程和UI设计。

## 项目位置

```
demo/yitu-demo/
```

## 快速启动

1. 进入项目目录：
```bash
cd yitu-demo
```

2. 安装依赖（如果还没安装）：
```bash
npm install
```

3. 启动开发服务器：
```bash
npm run dev
```

4. 在浏览器中访问：http://localhost:5173/

## 已实现的功能模块

### ✅ 1. 首页 (Home)
- 轮播图广告位
- 6大功能入口（赛事报名、AI对弈、人人对弈、围棋习题、教学视频、围棋商城）
- 最新动态展示
- 底部导航栏

### ✅ 2. 登录注册 (Login)
- 微信授权登录模拟
- 手机号验证码登录
- 用户注册流程
- 登录/注册切换

### ✅ 3. 赛事运营系统
**赛事列表页 (Events)**
- 三个标签页：报名中、即将开始、已结束
- 赛事卡片展示（图片、标题、时间、地点、人数、费用）
- 赛事标签（线上赛、线下赛等）

**赛事详情页 (EventDetail)**
- 赛事基本信息
- 时间地点详情
- 比赛规则说明
- 奖项设置
- 赛程安排
- 报名表单弹窗（姓名、手机、身份证、性别、年龄、棋力等级）

### ✅ 4. 学习系统
**学习主页 (Learning)**
- 用户学习统计（虚拟币、等级、胜率、对局数）
- 4个学习模块卡片（AI对弈、人人对弈、围棋习题、教学视频）
- 最近活动记录
- 虚拟币获取说明

**AI对弈 (AIGame)**
- 27个难度等级选择（18k到9d）
- 对弈界面展示
- 悔棋、认输、结束对弈功能

**人人对弈 (PVPGame)**
- 匹配对手流程
- 对战界面
- 认输、申请和棋功能

**围棋习题 (Practice)**
- 做题统计（已做题数、正确题数、正确率）
- 题目类型选择（死活题、定式题、手筋题、布局题）
- 6个难度等级（入门、初级、中级、高级、低段、高段）
- 错题集入口

**教学视频 (Videos)**
- 视频分类（入门教程、基础提升、中级进阶、高级实战、专题讲座）
- 视频列表展示（缩略图、标题、时长、作者、观看数）
- 视频播放模拟

### ✅ 5. 宣传栏
**新闻列表 (News)**
- 分类标签（全部、赛事动态、文化活动、政策文件）
- 图文新闻卡片（图片、标题、摘要、日期、阅读数）

**新闻详情 (NewsDetail)**
- 头图展示
- 文章标题和元信息
- 富文本内容展示

### ✅ 6. 商城系统
**商品列表 (Shop)**
- 商品分类（全部、围棋器具、文创周边、学习耗材）
- 商品卡片（图片、名称、销量、现金价格、虚拟币价格）

**商品详情 (ProductDetail)**
- 商品图片展示
- 价格信息（现金/虚拟币）
- 商品描述
- 商品规格
- 购买/兑换按钮
- 支付确认弹窗

**订单管理 (Orders)**
- 订单列表（订单号、商品信息、价格、支付方式、状态）
- 订单状态（待发货、已发货、已完成）
- 物流信息
- 订单操作（查看详情、确认收货、再次购买）

### ✅ 7. 个人中心 (Profile)
- 用户信息展示（头像、昵称、等级、身份）
- 虚拟币余额
- 学习数据统计（对局数、胜率、做题数、学习时长）
- 功能菜单
  - 我的赛事（带徽章提示）
  - 学习记录
  - 我的订单（带徽章提示）
  - 个人信息
  - 消息通知
  - 帮助中心
  - 联系客服
- 退出登录

## UI设计特色

### 色彩方案
- **主色调**: 深绿色 (#2c5f2d) - 围棋文化的优雅气质
- **辅助色**: 橙色 (#ff8c00) - 赛事竞技的活力
- **强调色**: 蓝色 (#4a90e2) - 学习成长的希望

### 设计风格
- 简洁现代的卡片式设计
- 圆角和阴影营造层次感
- 流畅的动画过渡效果
- 清晰的信息层级
- 友好的交互反馈

### 交互体验
- 底部固定导航栏，方便快速切换
- 卡片悬停效果，提升交互感
- 模态弹窗，避免页面跳转
- 空状态页面，友好提示
- 加载动画，优化等待体验

## 技术实现

### 技术栈
- Vue 3 (Composition API)
- Vue Router 4
- Vite
- CSS3 (CSS Variables)

### 项目结构
```
yitu-demo/
├── src/
│   ├── views/          # 15个页面组件
│   ├── router/         # 路由配置
│   ├── App.vue         # 根组件（含顶部栏和底部导航）
│   ├── main.js         # 入口文件
│   └── style.css       # 全局样式
├── public/
├── package.json
└── vite.config.js
```

### 响应式设计
- 使用CSS媒体查询适配不同屏幕
- 移动端优先的设计理念
- 灵活的网格布局
