# 弈途微信小程序 Demo

这是一个基于 Vue 3 + Vite 开发的弈途微信小程序演示项目，主要用于向客户展示交互流程和UI设计。

## 项目概述

弈途是一个围棋文化推广平台，以"万州-围棋村超"项目为核心应用场景，提供赛事运营、学习系统、宣传栏、商城等功能。

## 技术栈

- **框架**: Vue 3 (Composition API)
- **构建工具**: Vite
- **路由**: Vue Router 4
- **样式**: CSS3 (CSS Variables)

## 功能模块

### 1. 首页
- 轮播图展示
- 功能导航入口（赛事、AI对弈、人人对弈、习题、视频、商城）
- 最新动态展示

### 2. 用户登录注册
- 微信授权登录模拟
- 手机号验证码登录
- 用户注册流程

### 3. 赛事运营系统
- **赛事列表**: 按状态筛选（报名中、即将开始、已结束）
- **赛事详情**: 时间地点、规则、奖项、赛程安排
- **报名功能**: 在线填写报名信息

### 4. 学习系统
- **学习主页**: 虚拟币展示、学习统计、最近活动
- **AI对弈**: 27个难度等级选择、对弈界面
- **人人对弈**: 智能匹配、实时对战
- **围棋习题**: 题目类型和难度选择、错题集
- **教学视频**: 分类浏览、视频播放

### 5. 宣传栏
- 图文新闻列表
- 分类筛选（赛事动态、文化活动、政策文件）
- 新闻详情页

### 6. 商城系统
- **商品列表**: 分类展示（围棋器具、文创周边、学习耗材）
- **商品详情**: 规格参数、价格展示
- **购买方式**: 现金购买、虚拟币兑换
- **订单管理**: 订单列表、状态跟踪

### 7. 个人中心
- 用户信息展示
- 学习数据统计
- 虚拟币余额
- 我的赛事、学习记录、订单
- 设置和帮助

## 设计特色

### 色彩方案
- **主色调**: 深绿色 (#2c5f2d) - 代表围棋文化的优雅
- **辅助色**: 橙色 (#ff8c00) - 突出赛事竞技性
- **强调色**: 蓝色 (#4a90e2) - 体现学习成长

### UI特点
- 简洁现代的卡片式设计
- 流畅的动画过渡效果
- 响应式布局，适配不同屏幕
- 清晰的信息层级
- 友好的交互反馈

### 用户体验
- 底部导航栏，方便快速切换
- 面包屑导航，清晰的页面层级
- 加载状态提示
- 空状态页面设计
- 模态弹窗交互

## 安装和运行

### 安装依赖
```bash
cd yitu-demo
npm install
```

### 启动开发服务器
```bash
npm run dev
```

访问 http://localhost:5173/ 查看demo

### 构建生产版本
```bash
npm run build
```

## 演示说明

### 数据说明
- 所有数据均为模拟数据，用于展示UI和交互流程
- 实际项目需要对接后端API

### 功能说明
- 围棋棋盘显示为占位符，实际项目需集成围棋引擎
- 支付功能为模拟流程，实际需对接微信支付
- 视频播放为模拟，实际需对接视频服务

### 交互流程
1. 首页浏览各功能模块
2. 点击登录体验注册/登录流程
3. 浏览赛事列表，查看详情并模拟报名
4. 进入学习系统，体验各学习模块
5. 查看宣传栏内容
6. 浏览商城，体验购买流程
7. 个人中心查看用户信息和数据
